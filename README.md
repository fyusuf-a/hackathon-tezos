## Quick Start
1. Clone the repo:
```bash
git clone https://github.com/fyusuf-a/hackathon-tezos.git
cd hackathon-tezos
```
2. Setup .env file

> [!NOTE]
> Stripe webhook functionality requires manual deployment to work properly.

3. Run docker compose
```bash
docker-compose up
```
